Soorten testen

 

Unit Testing

  • Onafhankelijk van externen (db, webservice, ...)
  • Snel!
  • Véél testen
  • Test Happy Path & Edge Cases
  • "Actieve vijand van de code"

Integration Testing

  • Test geïntegreerd met externen (db, webservice, ...)
  • Test integratie twee verschillende lagen
  • Trager dan unit tests
  • Minder test cases
